# Insurance Risk Analytics

This project analyzes historical insurance data to identify risk patterns, evaluate profitability, and prepare for hypothesis testing and predictive modeling.

## Business Objective
AlphaCare Insurance Solutions wants to identify low-risk segments and improve premium pricing using data-driven insights.

## Project Scope
- Task 1: Exploratory Data Analysis
- Task 2: Data Version Control (DVC)

## Folder Structure
- `notebooks/` – EDA and pipeline notebooks
- `src/` – reusable Python modules
- `scripts/` – executable scripts
- `data/` – raw and processed datasets
- `reports/` – interim and final reports
